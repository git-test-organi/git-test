# git-test

이 프로젝트는 Github에서 Repository를 사용할 때 관리해야 할 것들을 연습하기 위한 repository 입니다.

[See Demo](https://github.com/git-test-organi/git-test)

## Prerequisite

- java: ^11

## How to Start

```shell
$ git clone {repo addr}
$ cd repo-setup-sample
$ npm install
$ npm start
```

- Board
- Admin page
